Benway

Control how many tabs you can open in a Firefox window

Maximum Tabs

Set the maximum number of tabs allowed in each window. When this limit is reached, new tabs will be automatically closed.

Tips for Using Benway

Finding Your Sweet Spot

Most users find that 5-10 tabs per window provides a good balance between productivity and browser performance.

Multiple Windows

Need more organization? Consider using multiple Firefox windows with different tab limits for different tasks.

When You Need More Tabs

The settings page can always be opened, even when you've reached your tab limit.
